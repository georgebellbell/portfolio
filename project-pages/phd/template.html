About This Project
This project investigates how immersive technologies like VR can help users understand and interact with complex, high-dimensional datasets. The focus is on intuitive interaction, visual clarity, and leveraging spatial metaphors for data exploration.
Technologies: Unity, C#, OpenVR, Custom Shaders
